## Deployment Instructions

### Quick Deploy (5 minutes)

1. **Create GitHub Account**
   - Visit https://github.com
   - Sign up (free)

2. **Upload Files**
   - Create repository: `jvc-emas-website`
   - Upload all files and folders

3. **Deploy on Netlify**
   - Go to https://www.netlify.com
   - Sign up with GitHub
   - Select repository
   - Click Deploy
   - Website is live!
